DynamoLauncher

Minecraft without complications, fast and stable.

About

Dynamo began as a personal endeavor initiated by myself, not_staff. I enjoyed working on it so much that
it's now more like my main project. It is a Minecraft Launcher made with Python. I wanted to
test the limits of the Python programming language and so here we are. Dynamo is a pretty
stable launcher, being a fork of another launcher, minecraft-launcher-lib. It is compatible with all Minecraft
versions. It even supports mods (Forge and Fabric).
